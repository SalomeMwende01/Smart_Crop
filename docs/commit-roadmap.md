# Commit Roadmap

This file tracks the commit strategy used for this project.

1. `chore: initialize FieldPulse monorepo structure`
2. `feat(backend): scaffold django project and core apps`
3. `feat(auth): implement role-based authentication API`
4. `feat(fields): add field models, updates, and status computation`
5. `feat(frontend): scaffold react app with routing and api client`
6. `feat(frontend): add auth flow and protected dashboards`
7. `feat(frontend): implement admin field management screens`
8. `feat(frontend): implement field agent update workflows`
9. `test: add backend tests and seed demo data`
10. `docs: finalize setup, design decisions, and deployment guide`
